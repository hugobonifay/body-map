# Interactive Body Map

Interactive SVG body map component for React.
Select, hover, and visualize body parts easily — perfect for sports, medical, or survey use cases.

## ✨ Features

- 🎯 Click to select body parts
- 🖱️ Hover interactions
- 🌍 Multi-language support (Français / English / Deutsch / Español / Italiano / Nederlands)
- 🔄 Anterior / Posterior views
- 🎨 Fully customizable styles and colors
- ⚛️ Lightweight, no external dependencies

## Installation

```
git clone https://github.com/hugobonifay/body-map.git
cd body-map
npm install
npm run dev
```

## 📊 Use Cases

- Sports performance tracking (fatigue, soreness)
- Medical / physiotherapy interfaces
- Surveys (“Where do you feel pain?”)
- Games / character stats
- Data visualization overlays

## 🧩 Structure

- SVG-based rendering
- Each body part is a `<path>`
- Fully declarative React API
